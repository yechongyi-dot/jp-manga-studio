# 背景图片库（AI 智能生成用）

两个文件夹，软件按**文件名顺序**自动轮选、**不重复**、选完**循环**（游标记在 `temp/bg-cursor.json`）：

- `top/`    顶部**静态短图**（不滚动）。建议统一规格，如 1080×320。
- `scroll/` **长图**（上下滚动）。建议 1080 宽、较高（如 1080×4000），越高滚动空间越大。

放很多张「规格一致、数据有差异」的图即可，软件每条视频依次取下一张。

## 开关 / 方向

`config/backgrounds.json`：
- `mode`: `image`（用图片库）/ `procedural`（用程序化背景，跟随口吻配色）
- `image.dir`: `up`（向上滚）/ `down`（向下滚）
- `image.scrimOpacity`: 遮罩不透明度（0~1，0=无遮罩，图上数据更清晰）

> 当前 `top/`、`scroll/` 内为占位测试图（纯色/渐变），换成你的真图即可。
